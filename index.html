A pagina da inicial será apenas para passar detalhes do projeto

Este projeto tem como objetivo o desenvolvimento de uma aplicação web voltada para uma cafeteria fictícia. A ferramenta permitirá aos usuários realizar pedidos e consultar a lista de pedidos efetuados de forma prática e intuitiva.
A interface do sistema será construída utilizando HTML e CSS, com o auxílio do framework Bootstrap para garantir um design responsivo e moderno. Para a manipulação dinâmica dos elementos da página, será utilizada a biblioteca jQuery, em substituição ao JavaScript puro, visando facilitar a implementação de funcionalidades interativas.
